...Proje tanıtımı: Arduino ile ölçülen sıcaklığı web sayfasında gösterme.

...Açıklama: 

	Projenin amacı server odasında ki sıcaklığı web de kullanıcı tarafına aktarmaktır. 
Birçok defa odanın soğutucularının durması ile odadaki sıcaklık yükselmiş ve sunucular çalışmayı durdurmuştur.
Bu amaçla arduino ile sıcaklık sensörü vasıtasıyla sıcaklık ölçmek ve belirlenen IP adresine XML formatında yazılarak 
sonuçta bir Web uygulaması ile bunu kullanıcı tarafına düzgün şekilde yansıtmaktır.
